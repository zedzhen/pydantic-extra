1.1.0 (22.06.2026)
------------------

Новые функции
+++++++++++++

* Новый метод :py:meth:`pydantic_extra.db.DB.setup_engine`.
  Он настраивает конкретный экземпляр :py:class:`sqlalchemy.Engine`.
* Добавлена поддержка альтернативных библиотек для :py:class:`~pydantic_extra.db.MySQL`
* Добавлена поддержка асинхронного использования баз данных :gh:`1`
* Новые группы зависимостей ``db_full``, ``adb_full``, ``full``

Устаревшие функции
++++++++++++++++++

* Метод :py:meth:`pydantic_extra.db.DB.setup` устарел и будет удалён в версии 2.0.
  Используйте :py:meth:`pydantic_extra.db.DB.setup_engine`.
* Удалена поддержка python 3.8, 3.9

* Добавлены тесты
