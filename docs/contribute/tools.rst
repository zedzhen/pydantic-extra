Инструменты
===========

tools/autobuild_docs.py
-----------------
Собирает документацию для просмотра по адресу `127.0.0.1:8000 <http://127.0.0.1:8000>`_.
Автоматически пересобирает документацию при обновлении файлов.

tools/build_docs.py
-----------------
Собирает документацию.


tools/gen_po.py
-----------------
Генерирует/обновляет po файлы для документации для указанных языков.

.. code:: console

  python3 tools/gen_po.py ru en

tools/gen_pot.py
-----------------
Генерирует/обновляет pot файлы для документации.
