Вклад
=====

pydantic-extra - это проект с открытым исходным кодом, поддерживаемый волонтёрами. Мы приветствуем любой вклад в проект.

Настройка
---------

* Склонируйте этот репозиторий

  .. code:: console

    git clone https://github.com/zedzhen/pydantic-extra.git
    cd pydantic-extra

* Создайте и активируйте виртуальное окружение

* Установите зависимости

  .. code:: console

    pip install --group dev --group autodocs

* Установить хук pre-commit

  .. code:: console

    pre-commit install
